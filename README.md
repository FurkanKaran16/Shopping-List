# 🛒 Shopping List App

A modern, dynamic, and responsive shopping list application built with **JavaScript**. This project demonstrates efficient DOM manipulation, state management, and persistent data storage.

> [!TIP]
> **[Click here to view.](https://furkankaran16.github.io/Shopping-List/)**

---

## 🚀 Key Features

- **Real-time Interaction:** Add, check, and remove items instantly without page reloads.
- **Persistent Storage:** Uses `LocalStorage` to ensure your list remains saved even after closing the browser or refreshing the page.
- **Advanced Filtering:** Filter items based on their status: *All*, *Completed*, or *Incomplete*.
- **In-place Editing:** Double-click on any active item to rename it directly.
- **Smart UI States:** The interface dynamically hides filters and the "Clear All" button when the list is empty, showing a friendly alert instead.
